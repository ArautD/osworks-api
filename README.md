# OSWorks 📄

Criação de uma API de gerenciamento de contrato,
ordem de serviços (OS), projetos,cliente e fornecedor 
juntamente com a
emissão de OS com base no contrato de serviço criado.

## Tecnologias Utilizadas 💻

|  Ferramenta   |  Versão  |
|:-------------:|:--------:|
|  Spring Boot  |  2.7.9   |
| Java Zulu JDK |  17.0.5  |
